# Firas Abadi - Portfolio

This is my personal portfolio, built to share the projects and skills I've developed.

I'm focused on learning new technologies and improving my front-end development practice.

## ⚙️ Technical Summary

* **Core Stack:** HTML5, Tailwind CSS, and Vanilla JavaScript.
* **Key Scripts:**
    * **Smooth Navigation:** JavaScript handles smooth scrolling and highlights the active section based on scroll position (using the Intersection Observer API).
    * **Animations:** Uses the Intersection Observer API for timed content reveals (`.fade-in`).
    * **Background:** Dynamic particle effect implemented with the HTML `<canvas>` element.
